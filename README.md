# CT&DT-Galgotias-Batch No:5 -Task 4 – User Persona

## 📋 Project Objective
This repository documents the User Persona created for the "Techniques of Empathy Building" module. The persona is designed to capture target customer insights, challenges, and motivations to ground product design decisions in authentic user needs.

## 👥 Persona Dimensions
The User Persona covers the following critical features:
* **Demographic Information:** Persona name, age, background, and personal details.
* **Goals & Objectives:** Key performance indicators and lifestyle motivations.
* **Psychographic Information:** Interests, personality traits, and behavioral choices.
* **Behavior & Preference:** Daily interaction habits and platform preferences.
* **User Journey:** The step-by-step path the persona takes to interact with our solution.
* **Challenges & Pain Points:** Specific obstacles and frustrations the user encounters.
